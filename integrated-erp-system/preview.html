üöÄ Quick Start Guide

Option 1: Docker Setup (Full Experience)

  1. Open PowerShell as Administrator
  2. Run: Set-ExecutionPolicy -ExecutionPolicy Bypass -Scope Process
  3. Navigate to: cd integrated-erp-system
  4. Run: .\scripts\quick-preview.ps1
  5. Wait 2-3 minutes for services to start
  6. Open: http://localhost:3000

Option 2: Manual Docker Commands

  1. Open Command Prompt or PowerShell
  2. Navigate to: cd integrated-erp-system
  3. Run: docker-compose -f docker/docker-compose.yml up -d
  4. Wait for services to start
  5. Open: http://localhost:3000
